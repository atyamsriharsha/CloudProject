        Real Time Event Monitoring System

Goal:To monitor the real time data which is very large and process in real time processing instead of Batch processing Which is Very useful these Days
and perform the Sentimental Analysis using Apache Spark

Environment Setup:Install Apache spark Locally or Install it on Docker and Download the nltk,yaml libraries in Python

Input: Given a Large Dataset Which is in the form of .txt  and enter the Post which U want to perform the Sentimental Analysis and U can also Know the most Trending Topics.We are performing the analysis using the Flickr Data

Output:Prints the Score of the Post on which sentimental Analysis Is Performed .Larger Value tells the post is more positive and the smaller Value implies the users found the post negatively
